# 🔍 Nmap Network Scanning Project

## 📌 Project Overview
This project demonstrates network reconnaissance and port scanning using Nmap in a Kali Linux environment.

---

## 🛠️ Tools Used
- Kali Linux
- Nmap

---

## ⚙️ Techniques Performed

### Host Discovery
nmap -sn 192.168.1.0/24

### Port Scanning
nmap 192.168.1.0/24

### Service Detection
nmap -sV 192.168.1.0/24

### Aggressive Scan
nmap -A 192.168.1.0/24

---

## 📊 Key Findings
- Active devices found in network
- Open ports: 53 (DNS), 80 (HTTP)
- Services detected: DNS, HTTP
- Some ports filtered (Telnet)
